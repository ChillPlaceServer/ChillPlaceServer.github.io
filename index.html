Bienvenue sur ChillPlace 🌍

Ce guide détaillé va t'accompagner pas à pas pour installer et configurer PollyMC, un launcher Minecraft cracké optimisé, afin de rejoindre le serveur privé entre amis dans la version 1.21.11.

Prends ton temps ! Lisez bien chaque étape sans vous précipiter. Les erreurs courantes viennent souvent d'une étape sautée (comme l'installation de Java).

Prérequis Système

Avant même de télécharger le jeu, il faut préparer ton ordinateur. Minecraft 1.21.11 a des exigences strictes.

☕ 1. Installer Java 21 (Obligatoire)

Pour la version 1.21.11, il est impératif d'installer Java 21. Sans cela, le jeu refusera tout simplement de se lancer ou affichera une erreur incompréhensible.

Même si tu as déjà Java 8 ou Java 17 installé, cela ne suffira pas. Tu dois impérativement avoir Java 21.

  1. Va sur le site officiel de téléchargement (Ex: Eclipse Adoptium).
    2. Télécharger Java 21
  2. Le fichier va se télécharger automatiquement, tu devrais avoir une interface similaire à celle sur l'image.
  3. Une fois le fichier .zip téléchargé, va dans ton explorateur de fichiers, puis entre dans le dossier .zip téléchargé.
  4. Clique sur le dossier nommé jdk-21.0.10+7-jre puis glisse-le en dehors du fichier .zip (tu peux voir une image d'illustration en dessous)
  5. Tu peux maintenant sortir du dossier .zip, et le supprimer en vérifiant que tu as bien maintenant un dossier appelé jdk-21.0.10+7-jre.

Ne supprime pas le dossier jdk-21.0.10+7-jre, sinon Minecraft ne se lancera pas. Garde-le dans un endroit que tu connais !

🎮 2. Pilotes Graphiques

Vérifie que les pilotes de ta carte graphique (NVIDIA, AMD ou Intel) sont à jour. Les nouvelles versions de Minecraft utilisent des moteurs de rendu récents qui peuvent faire planter un PC avec de vieux pilotes. Pour vérifier que tes pilotes sont à jour, tu peux utiliser l'application GeForce Experience pour les cartes graphiques NVIDIA, AMD Radeon Software pour les cartes graphiques AMD et Intel Driver & Support Assistant pour les cartes graphiques Intel. Les ordinateurs fournis par le lycée Alain Chartier sont déjà à jour sur ce sujet, tu peux passer cette étape. Si tu bloques ici, tu peux contacter TurtleSmile sur discord.

  2. Sélectionne "Extraire tout..." (ou utilise WinRAR / 7-Zip si tu l'as).
  3. Choisis un endroit sûr (comme ton Bureau ou tes Documents) en cliquant sur Parcourir.
  4. Lance l'extraction en cliquant sur extraire

Configuration initiale de PollyMC

Rends-toi dans le dossier que tu viens d'extraire. Tu y trouveras un fichier nommé pollymc.exe. Double-clique dessus pour le lancer.

  1. Une fenêtre de bienvenue apparaît. Clique sur Browse, puis entre dans le dossier jdk-21.0.10+7-jre téléchargé plus tôt.
  2. Dans le dossier ouvert, va dans bin, puis sélectionne javaw.exe.
  3. Laisse les paramètres de mémoire (RAM) par défaut, à moins que tu saches ce que tu fais (4096 Mo est un bon début).
  4. Clique sur Terminer.
  5. La fenêtre par défaut de PollyMC devrait s'ouvrir sous quelques secondes.

Création d'un profil "Crack" (Hors-ligne)

Pour jouer sans compte officiel acheté, nous allons créer un compte local (Hors-ligne).

  1. En haut à droite du launcher PollyMC, clique sur le menu Comptes, puis sur Configurer les comptes.
  2. Clique sur le bouton Ajouter un compte hors-ligne (ou Offline account).
  3. Une petite fenêtre apparaît. Saisis ton pseudonyme. Quand tu as fini, clique sur OK.
  4. Tu peux maintenant voir ton compte dans la liste. Clique sur Close.

Ce nom sera celui affiché dans le chat et au-dessus de ta tête en jeu. Choisis bien !

Installation de l'instance 1.21.11

Une "instance" est une installation de Minecraft. PollyMC permet d'en avoir plusieurs. Créons celle pour notre serveur.

  1. Dans le menu principal, clique sur le gros bouton Ajouter une instance (en haut à gauche).
  2. Dans la liste des versions, assure-toi d'être dans l'onglet "Personnalisé", que la catégorie "Filtre" est sur "Versions officielles", que "Chargeur de mods" est sur "Aucun" et sélectionne la version 1.21.11.
  3. Donne-lui un nom sympa, comme "ChillPlace Serveur".
  4. Clique sur OK.

🧩 Optionnel : Installer des Mods (Fabric)

Si ton PC manque de puissance ou si tu veux des shaders, tu peux utiliser Fabric :

  • Fais un clic droit sur ton instance fraîchement créée -> Modifier....
  • Va dans l'onglet Version et clique sur Installer le Loader sur la droite, puis sur Fabric. Laisse la version recommandée et valide.
  • Tu pourras ensuite ajouter les mods que tu souhaites

Attention ! Cette partie n'est pas obligatoire, je ne donne donc pas plus de détails !

Lancement du jeu et Connexion

C'est l'heure de vérité !

  1. Double-clique sur ton instance "ChillPlace Serveur" dans PollyMC.
  2. Le launcher va télécharger automatiquement tous les fichiers du jeu depuis les serveurs de Mojang (ça peut prendre quelques minutes selon ta connexion).
  3. Une fois terminé, le menu principal de Minecraft s'ouvre.
  4. Désactive le narrateur (Narrator: OFF), change la langue si besoin (Language...) et clique sur Continue.
  5. Te voilà sur la page d'accueil de Minecraft, tu peux cliquer sur Options... si tu as besoin de changer des paramètres, et appuie ensuite sur Multiplayer.
  6. La première fois, une popup de confirmation apparaît, appuie sur Do not show this screen again puis sur Proceed.
  7. Clique ensuite sur Add Server.
  8. Remplis les champs comme suit :
    • Nom du serveur : ChillPlace
    • Adresse du serveur : 
      Bientôt disponible...
  9. Appuie sur Done, puis sélectionne le serveur apparu et appuie sur Join Server.
  10. Te voilà enfin sur l'écran de connexion vers notre serveur, bienvenue !
